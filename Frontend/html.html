HTML

HTML means Hyper Text Markup Language. It is one kind of a markup language. Text could be edited by using Microsoft Word,Word pad, and Notepad, but documents created by these programme can not be published on the web.Hence, to convert a text document created by these programme to web compatible standards a markup language is required. More on HTML here-w3schools .According to the W3 school “HTML is a language for describing web pages.” In addition, HTML is not a programming language such as-visual basic and C++.HTML is a script language. In HTML,without using image,audio,video or animation a link could easily be created. HTML is written by using “Tag”. It is possible to create webpage by using a small number of Tag. HTML documents are very small,therefore it is transferred quickly within Net.
